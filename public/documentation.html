Business Logic URL HTTP Method Query Parameters Request body data format Response body data format
Return a list of ALL movies to the user /movies GET none none A JSON object holding data about all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/:name GET name (URL parameter) none A JSON object holding data about a single movie including description, genre, director, image URL. Example:
{
  title: 'Star Wars: Episode 1',
  description: 'A movie about Jedi and Sith',
  genre : 'Sci-Fi',
  director: 'George Lucas',
  imageUrl: 'https://www.exampleimage.com'
}
Return data about a genre (description) by name/title (e.g., “Thriller”) /genres/:name GET name (URL parameter) none A JSON object holding data about a single movie including description, genre, director, image URL. Example:
{
  name: 'Thriller',
  description: 'Thrillers are very thrilling'
}
Return data about a director (bio, birth year, death year) by name /directors/:name GET name (URL parameter) none A JSON object holding data about a director including bio, birth year and death year. Example:
{
  name: 'George Lucas',
  bio: 'A talented director and founder of the best Sci-Fi universe.',
  birth year: '05/14/1944',
  death year: 'alive'
}

} 		A confirmation or an error message.

{
  username: 'Nic Kelback'
} 		A confirmation message and updated user data or an error message. Example:
{

Allow users to add a movie to their list of favorites /users/:username/favorites POST username (URL parameter) A JSON object holding data about a favorite movie. Example:
{
  title: 'Peter Jackson's King Kong'
} 		A confirmation or an error message.
Allow users to remove a movie of their list of favorites /users/:username/favorites/:movieTitle DELETE username (URL parameter), movieTitle (URL parameter) A JSON object holding data about a favorite movie to remove. Example:
{
  title: 'Peter Jackson's King Kong'
} 		A confirmation or an error message.
Allow existing users to deregister /users/:username/ DELETE username (URL parameter) none A confirmation or an error message.
